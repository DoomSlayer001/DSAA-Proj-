核心算法实现
 Intelligent Scissors

 解决平均多少点进行一次重置的问题，及根据分辨率计算经过多少像素点后进行一次最优路径计算（可根据像素固定），再经过几轮最优统一后若最优统一则建立新起始点（灵活变动）
 根据图像分辨率（具体应该是当前进行缩放后正在进行操作的图像的分辨率来调整鼠标监听的频率及新起始点重置的频率，感觉应该除以分辨率但不能过小）

 考虑在套索闭合后对内部图像进行二次优化
